# Next Contact App

Simple Next.js contact form with a backend API that appends submissions to `submissions.json`.

Usage locally:

1. Clone
   ```bash
   git clone https://github.com/debabratasahu30/next-contact-app-stitch-20260601.git
   cd next-contact-app-stitch-20260601
   npm install
   npm run dev
   ```

2. Open http://localhost:3000 and submit the form.

Notes:
- Writing to `submissions.json` is fine for demos, but not persistent on many hosting providers. For production use a database (Postgres, MongoDB, Supabase, etc.).

Deploy (free):
- Push this repo to GitHub (already pushed).
- Deploy to Vercel: connect your GitHub account and import this repository. Vercel will build and run the Next.js app.
